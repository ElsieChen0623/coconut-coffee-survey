# Coconut Coffee Survey

## Project Overview
This project aims to gather feedback on coconut coffee and its various flavors. The survey will help us understand customer preferences and improve our product offerings.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ElsieChen0623/coconut-coffee-survey.git
   ```
2. Navigate into the project directory:
   ```bash
   cd coconut-coffee-survey
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
